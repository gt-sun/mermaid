# mermaid
Let's mermaid

**知识点**：

- `graph TD`：TD 表示从上到下 (Top-Down)，LR 表示从左到右 (Left-Right)。 Instead of `flowchart` one can also use `graph`.
- `[]`：圆角矩形（普通节点）。
- `()`：圆角椭圆（开始/结束）。
- `{}`：菱形（判断/决策）。
- `-->`：实线箭头。
- `-.->`：虚线箭头（通常表示异步或可选）。
- `subgraph 标题` ... `end`：定义一个区域。
- `direction LR`：可以在子图内部单独指定方向。
- `id[文本]`：给节点定义 ID，方便后续引用样式。
- `==>`：加粗箭头，强调主路径。
- `style`：自定义节点颜色（红色代表故障，绿色代表正常）。
- `:::`：应用样式类。
- `classDef`：定义样式类（颜色、边框、字体）。
- 特殊形状：`[(数据库)]`, `[[配置文件]]`, `[/日志文件/]`。
